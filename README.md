# 🔐 Сканер паролей на утечки

Утилита для проверки паролей по базе известных утечек через API сервиса HaveIBeenPwned.

## Что умеет
- Проверяет пароль по базе более 800 млн скомпрометированных паролей
- Оценивает надёжность пароля по 5 критериям
- Показывает советы по улучшению пароля
- Графический интерфейс

## Как работает
Программа не отправляет пароль в интернет напрямую. Используется метод k-Anonymity:
пароль хешируется через SHA-1, и на сервер отправляются только первые 5 символов хеша.
Это защищает конфиденциальность пользователя.

## Технологии
- Python 3
- tkinter (интерфейс)
- requests (запросы к API)
- hashlib (SHA-1 хеширование)

## Запуск
pip install requests
python checker.py
